1. Objective
The goal of this project is to build a Smart Task Analyzer. This is a miniapplication that takes a list of tasks (with properties like due date, importance, and
effort) and prioritizes them using a custom "intelligence" algorithm.
Your Primary Goal: Help a user decide what to work on first.
Key Focus: Show off your problem-solving skills, code quality, and ability to
handle edge cases (like missing data or past due dates).

2. Technical Requirements
Before starting, ensure you have the following installed and set up:

Software & Tools
Language: Python 3.8 or higher * Framework: Django 4.0 or higher *
Frontend: HTML5, CSS3, Vanilla JavaScript (No React/Angular required)
Database: SQLite (Default with Django, no extra installation needed)
Version Control: Git (for GitHub submission)
IDE/Editor: VS Code, PyCharm, or Sublime Text

Hardware/Environment
A computer capable of running a local server (localhost).
Internet access (for installing packages).
